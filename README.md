## Install

Installation on macOS requires these commands first:

    sudo softwareupdate -i -a
    xcode-select --install

### Clone with Git

    git clone https://github.com/mgm702/dotfiles.git ~/.dotfiles
    cd ~/.dofiles
    . bootstrap/install.sh
